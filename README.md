# Atividade 2 - Idade Média

Calcula a idade média de um grupo e classifica em: jovem, adulta ou idosa.

## Como funciona:
- Coleta idade de N pessoas
- Calcula a média
- Classifica: 0-25 (jovem), 26-60 (adulta), 60+ (idosa)

## Como usar:
```bash
python atividade2.py
```

## Exemplo:
```
Quantas pessoas são? 3
Idade de 1 pessoa: 20
Idade de 2 pessoa: 25
Idade de 3 pessoa: 30
Idades: [20, 25, 30]
A idade media é 25.00 anos.
É uma turma adulta
```
